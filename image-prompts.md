# Who's Better? - Image Generation Prompts

This file contains all vote items with their visual specifications and image generation prompts.

---

## Image Specifications

**Standard Size:** 400x400 pixels
**Format:** PNG
**Style:** Modern, clean, bold typography
**Font:** Sans-serif (Arial, Helvetica, or similar)
**Text Color:** White (#ffffff)
**Text Weight:** Bold
**Text Position:** Centered
**Background:** Solid color as specified below
